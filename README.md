## Nathan's DevCard - Stufe 1
Welcome to the first step of my Android development journey! This project is a Digital Business Card built to introduce and practice the fundamentals of Android Studio, Kotlin, and XML layouts.

##  The Project
My DevCard is a simple functional Android application that introduces my profile, my technical skills (Kotlin & Python), and provides direct links to my professional networks.

### 🛠 Features implemented so far (although not perfected:
- Dynamic Theme Toggling: A button to switch the app's visual theme.

- Language Selection: A PopupMenu implementation that allows users to choose between different languages (German, Portuguese, English and possibly more).

- Social Integration: Functional buttons that redirect to GitHub and LinkedIn profiles.

- Biometric Bio: A section that would be dedicated for a personal biography.
